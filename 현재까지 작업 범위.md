# ✅ 현재 완성된 핵심 업무 흐름

```text

    ↓

    ↓

    ↓
프레임 자동 추출

    ↓

    ↓
라벨 수정 또는 삭제
```

# 🎯 향후 개발 방향

## Priority 1 - 안정화

### 시스템 안정성 강화

* 업로드 용량 제한 검증
* 프레임 추출 간격 검증
* 잘못된 영상 파일 업로드 예외 처리
* DB 초기화 없이 마이그레이션 적용
* 프레임 이미지 로딩 실패 처리
* API 예외 처리 표준화
* 파일 저장 경로 검증
* JWT 만료 처리 개선

### 목표

> 실제 사용자 환경에서도 오류 없이 안정적으로 동작하는 서비스 구축

---

## Priority 2 - 라벨링 UX 개선

### 사용자 편의성 향상

* 프레임 전체 화면 보기
* 라벨별 필터링
* 미분류 프레임만 조회
* 다음 프레임 이동
* 이전 프레임 이동
* 라벨 단축키 지원

### 예시

| 단축키 | 동작       |
| --- | -------- |
| F   | Fire     |
| S   | Smoke    |
| C   | CarLight |
| N   | Negative |
| ←   | 이전 프레임   |
| →   | 다음 프레임   |

### 목표

> 대량 데이터 라벨링 속도 향상

---

## Priority 3 - AI 자동 분류 연동

### AI Classifier 구축

* classifier_service.py 구현
* 모델 로드 기능
* 자동 예측 API 구현

### 자동 분류 프로세스

```text
영상 업로드
    ↓
프레임 추출
    ↓
AI 자동 예측
    ↓
Fire Confidence
Smoke Confidence
CarLight Confidence
Negative Confidence 저장
    ↓
사용자 검수
    ↓
최종 라벨 확정
```

### 목표

> 수동 라벨링 시간을 크게 단축

---

## Priority 4 - Export 기능

### 데이터셋 내보내기

* YOLO Format Export
* COCO Format Export
* 라벨링 완료 데이터만 Export
* ZIP 다운로드

### 지원 예정 형식

```text
exports/

├── yolo/
│   ├── images/
│   └── labels/
│
└── coco/
    ├── images/
    └── annotations.json
```

### 목표

> AI 학습에 바로 사용할 수 있는 데이터셋 생성
