# ⚠️ 가장 많이 발생할 수 있는 오류

## 1. CORS 문제

대표 증상

```text
Access-Control-Allow-Origin
```

오류 발생

확인 사항

```python
CORS(app)
```

설정 여부 확인

---

## 2. JWT 토큰 저장 문제

확인 항목

```text
localStorage 저장 여부
Authorization 헤더 포함 여부
```

---

## 3. API 응답 타입 불일치

예시

```typescript
response.data.user
```

실제 응답

```json
{
  "data": {
    "user": {}
  }
}
```

형태 차이 확인

---

## 4. Import 경로 문제

예시

```typescript
../../
```

상대경로 충돌

권장

```typescript
@
```

Alias 사용

---

## 5. React Router 문제

확인 항목

```tsx
<Outlet />
```

누락 여부

---

## 6. SQLite 테이블 누락

확인

```powershell
python run.py
```

실행 시 테이블 생성 여부 확인

---

## 7. bcrypt / passlib 문제

설치 확인

```powershell
pip install bcrypt passlib
```
