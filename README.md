# 🚀 Dataset Classifier Tool

> **AI 학습용 이미지·영상 데이터를 빠르게 분류하고 라벨링하는 웹 기반 데이터셋 제작 플랫폼**

---

## 📌 프로젝트 개요

Dataset Classifier Tool은 AI 모델 학습에 필요한 이미지 및 영상 데이터를 업로드하고, 분류, 라벨링, 검수, 내보내기까지 수행할 수 있는 데이터셋 제작 보조 플랫폼입니다.

초기 목표는 도로, 터널, 교통사고, 화재, 연기, 차량 등화류와 같이 오탐 가능성이 높은 교통 환경 데이터를 효율적으로 수집하고 정리하는 것입니다.

향후에는 제조업, 안전관리, 스마트시티, 의료, 농업 등 다양한 산업 분야의 데이터셋 구축 작업으로 확장할 수 있도록 설계되었습니다.

---

## 🎯 프로젝트 목적

AI 모델 개발 과정에서 가장 많은 시간과 비용이 소모되는 데이터 전처리 과정을 자동화하여 다음과 같은 작업을 효율적으로 수행할 수 있도록 지원합니다.

* 데이터 수집
* 데이터 정리
* 데이터 분류
* 데이터 검수
* 데이터셋 생성

개인 개발자부터 연구팀, 기업까지 누구나 학습용 데이터셋을 빠르게 구축할 수 있는 환경을 제공합니다.

## 🛠️ MVP 범위

현재 MVP에서는 다음 기능을 우선 구현합니다.

### 1. 이미지 / 영상 업로드

사용자는 학습에 사용할 이미지 및 영상을 업로드할 수 있습니다.

### 2. 프레임 추출

업로드된 영상은 프레임 단위로 분할되어 데이터셋에 저장됩니다.

### 3. 프로젝트 생성

데이터셋을 프로젝트 단위로 관리할 수 있습니다.

### 4. 수동 라벨링

사용자는 이미지에 직접 클래스를 지정하고 수정할 수 있습니다.

### 5. AI 자동 분류

반복적인 라벨링 작업을 줄이기 위해 AI 기반 자동 분류 기능을 제공합니다.

### 6. 데이터셋 내보내기

최종 데이터셋을 YOLO 또는 COCO 형식으로 다운로드할 수 있습니다.
